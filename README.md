# calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Basic arithmetic operations (+, -, *, /, %)
* Clean UI design
* Right-aligned display (like real calculator)
* Smart overflow handling (latest digits always visible)
* AC (clear) and DEL (delete) functionality

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript
